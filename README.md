# AntiAFK

Windows-приложение для GTA V multiplayer. Имитирует активность на маркетплейсе, чтобы персонаж не отключился по AFK.

**Скачать:** [Releases](https://github.com/BxdiS/antiafk/releases) → `AntiAFK-win-Setup.exe`

## Быстрый старт

1. Установи через Setup.exe из Releases.
2. Запусти — иконка появится в трее.
3. **Запустить** — старт бота. **Остановить** — пауза с сохранением состояния.

| Иконка | Статус |
|--------|--------|
| 🟢 | Работает |
| 🟡 | Ожидание игры |
| 🔴 | Остановлен |
| 🔵 | Доступно обновление |

Трей: **Настройки** · **Открыть лог** · **Обновить** (при апдейте) · **Выход**

## Требования

- Windows 10/11 x64
- Оконный режим без рамки **16:9**
