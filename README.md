# 🎮 Marketplace Anti-AFK | Маркетплейс Anti-AFK

## 🇷🇺 Русский

### Описание
Автоматический скрипт для Majestic Multiplayer, который предотвращает отключение персонажа по неактивности (AFK) путём имитации активной игры на маркетплейсе.

### Требования
- **Python 3.8+**
- **Windows** (требуется доступ к Windows API)
- Установленные зависимости (см. `requirements.txt`)

### Установка и запуск

1. **Клонируй репозиторий:**
   ```bash
   git clone https://github.com/BxdiS/marketplace-antiafk.git
   cd marketplace-antiafk
   ```

2. **Установи зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Запусти скрипт:**
   ```bash
   python main.py
   ```

### Как это работает
- ✅ Автоматически находит окно игры (GTA V / Majestic Multiplayer)
- ✅ Имитирует клики по категориям маркетплейса
- ✅ Генерирует случайные действия для избежания детекции
- ✅ Автоматически закрывает уведомления и предупреждения
- ✅ Сохраняет корректное состояние интерфейса

### Внимание ⚠️
- Используй на свой риск
- Может быть обнаружено системой защиты сервера
- Требуется запуск с правами администратора

## 🇬🇧 English

### Description
Automatic script for Majestic Multiplayer that prevents character disconnect due to inactivity (AFK) by simulating active gameplay on the marketplace.

### Requirements
- **Python 3.8+**
- **Windows** (requires Windows API access)
- Installed dependencies (see `requirements.txt`)

### Installation & Launch

1. **Clone the repository:**
   ```bash
   git clone https://github.com/BxdiS/marketplace-antiafk.git
   cd marketplace-antiafk
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the script:**
   ```bash
   python main.py
   ```

### How It Works
- ✅ Automatically finds the game window (GTA V / Majestic Multiplayer)
- ✅ Simulates marketplace category clicks
- ✅ Generates random actions to avoid detection
- ✅ Auto-closes notifications and warnings
- ✅ Maintains correct UI state

### Warning ⚠️
- Use at your own risk
- May be detected by server protection system
- Requires administrator privileges to run
